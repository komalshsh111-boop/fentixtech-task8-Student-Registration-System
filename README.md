# 🎓 Student Registration System

A simple Student Registration System built using **HTML, CSS, Bootstrap 5, and JavaScript**.  
This is my first JavaScript project as part of the FentixTech Internship (Task #08), covering DOM manipulation, form validation, and dynamic data handling.

## 📋 Features

- Registration form with fields: Full Name, Email Address, Phone Number, Course Name
- Real-time form validation using JavaScript (no external libraries)
- Error messages shown for invalid inputs
- Form submission blocked until all fields are valid
- Registered students displayed in a dynamic table
- Edit button to update existing student records
- Delete button to remove records (with confirmation prompt)
- Fully responsive UI using Bootstrap 5

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS / DOM Manipulation)
